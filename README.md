##使用

electron

vue

websocket

需要后端配合[go-server](https://github.com/bbfpl/go-server "go-server")

登录走普通接口/login ,登录成功后获取token(uid和name构成)

客户端缓存保存uid和name


websocket的onmessage接收数据

根据type的不同做不同的处理

暂时定义了 msg,userJoin,userOut

msg 消息

userJoin 有新用户加入

userOut 用户退出
